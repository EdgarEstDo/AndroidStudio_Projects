# 📱 Android Studio Projects: My Dev Bookmark

¡Bienvenido/a a mi bitácora de desarrollo Android! Este repositorio funciona como mi **marcador personal (bookmark)**, donde centralizo y documento todos los proyectos, prácticas y experimentos que voy realizando en Android Studio.

El objetivo es mantener un historial organizado de mi evolución técnica, desde conceptos básicos de arquitectura hasta integraciones avanzadas.

---

### 🚀 ¿Qué encontrarás aquí?

Cada carpeta de este repositorio representa un hito en mi aprendizaje o un módulo de aplicación funcional:

* **Proyecto Iniciales:** Estructura base y primeros pasos en el desarrollo.
* **Pruebas de Conexión:** Testeo de comunicación entre dispositivos y entornos.
* **UI & UX:** Experimentos con layouts, componentes visuales y manejo de pantalla.
* **Gestión de Datos:** Prácticas con persistencia y lógica de negocio.

---

### 🛠️ Stack Tecnológico

Estas son las herramientas y tecnologías principales que utilizo en mis proyectos:

* **Lenguaje:** Kotlin / Java.
* **IDE:** Android Studio.
* **Arquitectura:** MVVM / Clean Architecture (en progreso).
* **Componentes:** Jetpack Compose, XML Layouts, Room, Navigation Component.
